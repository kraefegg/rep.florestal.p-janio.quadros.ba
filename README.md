# Dashboard PRAD — UFV Apolo Presidente Jânio Quadros/BA

Dashboard executivo interativo (HTML/Chart.js) do monitoramento do plantio compensatório,
cercamento e sistema de irrigação — UFV Apolo Presidente Jânio Quadros/BA, Junho/2026.
Bragagnoli Ambiental (CTF/AIDA 8805586).

## Deploy (GitHub Pages)
1. Criar repo → subir `index.html`.
2. Settings → Pages → Branch: `main` /root → Save.
3. URL: `https://<usuario>.github.io/<repo>/`

## Conteúdo
- Visão Geral (KPIs, indicadores de monitoramento, painel comparativo, desempenho ambiental)
- Monitoramento das Mudas (espécies nativas, desenvolvimento vegetativo, campanhas, comparativo Abril × Maio × Junho)
- Indicadores Ambientais (Tabelas 6.3, 6.5, 6.5.1, semáforo ambiental)
- Avaliação Fitossanitária (herbivoria, doenças, sistema de irrigação)
- Execução do Programa (ações de manejo, área monitorada, coordenadas UTM, cronograma de 24 campanhas)
